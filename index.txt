Статус коды
/====/
200 - Код ответа об успешном статусе "The HTTP 200 OK" указывает, что запрос выполнен успешно.
Успешный результат PUT или DELETE

201 Created - код ответа на успешный статус означает, что запрос выполнен успешно и привёл к реальному ресурсу.
Новый ресурс эффективно создается до ответа на этот ответ.
И новый ресурс возвращается в телесообщение, его значение представляет собой либо запрос URL-адреса, либо заголовок запроса.
(Результатом запроса является общий случай использования этого кода состояния POST.)


/====/
300 Multiple Choices - Код ответа на перенаправление "The HTTP 300 Multiple Choices" указывает, что запрос имеет несколько возможных ответов. Пользователь-агент или пользователь должны выбрать один из них. Поскольку стандартного способа выбора одного из ответов нет, этот код ответа очень редко используется.
Используется для проверки на формат использования!!!!

301 Moved Permanently - (перенос страницы на другую ссылку) Код перенаправления 301 Moved Permanently протокола передачи гипертекста (HTTP) показывает, что запрошенный ресурс был окончательно перемещён в URL, указанный в заголовке Location. Браузер в случае такого ответа перенаправляется на эту страницу, а поисковые системы обновляют свои ссылки на ресурс (говоря языком SEO, вес страницы переносится на новый URL-адрес).

302 Found - Если страница или ресурс временно недоступны по привычному URL, но ожидается, что они вскоре вернутся на старый адрес, используется 302 Found.

/====/
400 Bad Request - Код состояния ответа "HTTP 400 Bad Request" указывает, что сервер не смог понять запрос из-за недействительного синтаксиса. Клиент не должен повторять этот запрос без изменений.

401 Unauthorized - Если пользователь пытается получить доступ к защищенной странице без входа в систему или с неправильным паролем, сервер может вернуть код 401 Unauthorized.

403 Forbidden - Отказ в авторизации: Код 403 Forbidden означает, что сервер принял и понял запрос, но отклоняет его выполнение из-за отсутствия у клиента соответствующих прав.

404 Not Found - Код ответа на ошибку HTTP 404 Not Found указывает, что сервер не может найти запрошенный ресурс. Ссылки, ведущие к коду 404,

405 Method Not Allowed - Сервер распознает метод, но он недоступен для конкретного ресурса.

429 oo Many Requests - Сервер установил ограничения на количество запросов, которые клиент может отправить за определенное время.
